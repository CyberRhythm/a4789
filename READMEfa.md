# ShadowLink 🔗

یه ابزار ساده و کاربردیه که بهت اجازه میده فایل‌هایی که توی تلگرام داری رو مستقیم بفرستی به بله — بدون دردسر، بدون نیاز به هیچ‌چیز پیچیده‌ای.

فایل رو توی تلگرام برای ربات بفرست، بقیه‌اش با ShadowLink‌ه.

> 📌 این پروژه مکمل پروژه اصلی [CyberRhythm](https://github.com/CyberRhythm) هست.
> راهنمای انگلیسی: [README.md](README.md)

---

## قبل از هر چیز — چی نیاز داری؟ 🛠️

قبل از نصب، به ۴ تا چیز نیاز داری که باید یادداشت کنی. یه دقیقه وقت میبره، ارزشش رو داره.

---

### ۱. شناسه عددی بله + ربات بله

**پیدا کردن شناسه عددی بله‌ات**

توی بله این ربات رو سرچ کن:

```
@Idbotbot
```

استارتش بزن، شناسه‌ات رو بهت میده. یادداشتش کن.

---

**ساخت ربات بله**

توی بله این ربات رو سرچ کن:

```
@botfather
```

> ⚠️ این ربات گاهی شلوغه و کند جواب میده — صبور باش، میارزه!

بعد از استارت:
1. روی **ساخت بازوی جدید** بزن
2. یه اسم دلخواه بده
3. یه آیدی یکتا انتخاب کن که حتماً به `bot` ختم بشه — مثلاً `myfilebot`
4. بعد از ساخت، روی **بازیابی توکن** بزن
5. یه چیزی شبیه این میگیری:

```
123456789:ABCdef-XYZghijk...
```

این توکن رو کپی کن و یادداشت کن.

---

### ۲. شناسه عددی تلگرام + ربات تلگرام

**ساخت ربات تلگرام**

توی تلگرام این ربات رو سرچ کن:

```
@BotFather
```

بعد از استارت:
1. دستور `/newbot` رو بزن
2. یه اسم و یه آیدی (که به `bot` ختم بشه) انتخاب کن
3. توکن ربات رو یادداشت کن

---

**پیدا کردن شناسه عددی تلگرامت**

توی تلگرام این ربات رو سرچ کن:

```
@myidbot
```

استارتش بزن، شناسه عددی‌ات رو بهت میده. یادداشتش کن.

---

### خلاصه — ۴ چیزی که نیاز داری 📋

| چی | از کجا |
|---|---|
| 🔵 شناسه عددی بله | ربات `@Idbotbot` در بله |
| 🔵 توکن ربات بله | ربات `@botfather` در بله |
| ✈️ توکن ربات تلگرام | ربات `@BotFather` در تلگرام |
| ✈️ شناسه عددی تلگرام | ربات `@myidbot` در تلگرام |

وقتی هر ۴ تا رو داشتی، بریم سراغ نصب!

---

## نصب 🚀

روی سرور اوبنتوت این دستور رو بزن:

```bash
bash <(curl -fsSL https://raw.githubusercontent.com/CyberRhythm/ShadowLink/main/tg2bale.sh)
```

اسکریپت ازت میپرسه:
1. توکن ربات تلگرام
2. شناسه عددی تلگرام
3. توکن ربات بله
4. شناسه عددی بله

بعدش همه چیز رو خودش نصب و راه‌اندازی میکنه — آپدیت سرور، نصب پیش‌نیازها، ساخت سرویس و همه چیز.

---

## منوی مدیریت ⚙️

بعد از نصب، هر وقت خواستی دوباره اسکریپت رو اجرا کن:

```bash
bash tg2bale.sh
```

از منو میتونی:

- **ربات رو شروع یا متوقف کنی**
- **ریستارت بدی**
- **لاگ زنده ببینی**
- **تنظیمات رو چک کنی**
- **در صورت نیاز دوباره نصب کنی**

---

## چطور کار میکنه؟ 💡

بعد از نصب، کافیه توی تلگرام فایل رو برای ربات بفرستی:

- یه فایل بفرستی → ربات دانلود میکنه و به بله میفرسته ✅
- چند فایل پشت هم بفرستی → صف میشن و یکی‌یکی به ترتیب ارسال میشن ✅
- فایل `.apk` یا `.exe` بفرستی → اسمشو تغییر میده که بله قبول کنه ✅
- ارسال موفق بشه → توی تلگرام بهت خبر میده با تاریخ و ساعت ایران ✅
- ارسال ناموفق باشه → تا ۳ بار دوباره تلاش میکنه ✅

---

## دستورات مفید 🖥️

```bash
# وضعیت ربات
systemctl status tg2bale-account1

# لاگ زنده
journalctl -u tg2bale-account1 -f

# ریستارت
systemctl restart tg2bale-account1
```
